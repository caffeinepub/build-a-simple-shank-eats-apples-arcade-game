# Exporting Shank Game as ZIP

This guide explains how to export your Shank Game as a single ZIP file that can be uploaded to PlayStory or any static hosting platform.

## Prerequisites

Before exporting, ensure you have:

- **Node.js** (v18 or higher)
- **pnpm** package manager installed
- All dependencies installed

## Automatic ZIP Generation

The production build process automatically generates a ZIP file of your game. No interactive prompts or confirmations are required.
